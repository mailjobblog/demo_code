| redis-c1  | 172.31.0.11 | 6301->6379 | 16301->16379  | master |

| redis-c3  | 172.31.0.13 | 6303->6379 | 16303->16379  | master |
| redis-c4  | 172.31.0.14 | 6304->6379 | 16304->16379  | slave |

| redis-c6  | 172.31.0.16 | 6306->6379 | 16306->16379  | slave |
